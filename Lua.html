Write a function that takes a table of numbers and returns the minimum, maximum, and average.

Answer:

Calculating Min, Max, and Average in Lua

  • Initialize variables for min, max, and sum.
  • Iterate through the table to find the minimum and maximum.
  • Calculate the sum of all numbers.
  • Calculate the average using the sum and the number of elements.
  • Return the minimum, maximum, and average values.

Code Example:

function calculateMinMaxAvg(numbers)
  local min = math.huge
  local max = -math.huge
  local sum = 0

  for _, num in ipairs(numbers) do
    if num < min then min = num end
    if num > max then max = num end
    sum = sum + num
  end

  local average = sum / #numbers

  return min, max, average
end

The function initializes min and max with extreme values, and sum with 0. It iterates through the numbers to find the minimum and maximum. It also calculates the sum. Finally, it calculates the average using the sum and returns the calculated values.

Question:

Write a function in Lua that merges two sorted arrays into a sorted array.

Answer:

Merge Sorted Arrays in Lua

  • Create an empty result array to store the merged elements.
  • Initialize variables 'i' and 'j' to iterate through the input arrays.
  • Compare elements at 'i' and 'j' and insert the smaller element into the result array.
  • Increment 'i' or 'j' based on which element was inserted.
  • Repeat the comparison and insertion until both arrays are exhausted.
  • If one array is exhausted, insert the remaining elements from the other array.
  • Return the merged sorted array as the result.
  • Note: This algorithm has a time complexity of O(n), where n is the total number of elements in the input arrays.

Code Example:

function mergeSortedArrays(arr1, arr2)
  local result = {}
  local i, j = 1, 1

  while i <= #arr1 and j <= #arr2 do
    if arr1[i] <= arr2[j] then
      table.insert(result, arr1[i])
      i = i + 1
    else
      table.insert(result, arr2[j])
      j = j + 1
    end
  end

  while i <= #arr1 do
    table.insert(result, arr1[i])
    i = i + 1
  end

  while j <= #arr2 do
    table.insert(result, arr2[j])
    j = j + 1
  end

  return result
end

The function initializes an empty 'result' array and two indices 'i' and 'j'. It then compares elements from 'arr1' and 'arr2', inserts the smaller element into 'result', and increments the corresponding index. Once an array is exhausted, the remaining elements are inserted into 'result'. The function returns the merged sorted array.

The code defines a metatable 'Stack' and a 'new' constructor to create a new stack object. The 'push' method inserts an element into the table, the 'pop' method removes and returns the last element, and the 'peek' method returns the last element without removing it.

How can you write a Lua script to parse a given CSV file into a table?

Answer:

Parsing CSV into a Table in Lua

  • Define a function 'parseCSV' that takes a string 'fileContent' (CSV data) as a parameter.
  • Initialize an empty table 'rows' to store the parsed data.
  • Iterate through each line of 'fileContent' using the pattern '[^\r\n]+'.
  • Inside the loop, initialize an empty table 'row' to represent a row of data.
  • For each line, iterate through each value using the pattern '[^,]+'.
  • Append each value to the 'row' table.
  • Append the 'row' table to the 'rows' table.
  • After parsing all lines, the 'rows' table contains the parsed CSV data.
  • The function returns the 'rows' table.
  • Time complexity: O(n)

Code Example:

function parseCSV(fileContent)
  local rows = {}
  for line in fileContent:gmatch("[^\r\n]+") do
    local row = {}
    for value in line:gmatch("[^,]+") do
      table.insert(row, value)
    end
    table.insert(rows, row)
  end
  return rows
end

local csvData = "Name,Age,Location\nAlice,25,New York\nBob,30,Los Angeles"
local parsedData = parseCSV(csvData)

for i, row in ipairs(parsedData) do
  print("Row", i)
  for j, value in ipairs(row) do
    print("Column", j, ":", value)
  end
end

The 'parseCSV' function uses pattern matching to iterate through lines and values of the CSV data. It constructs a 'row' table for each line and appends values to it. The 'row' tables are then appended to the 'rows' table, forming a table of parsed data.

Answer:

Fibonacci Sequence in Lua

  • For the iterative approach, use a loop to calculate Fibonacci numbers iteratively.
  • For the recursive approach, use a function that calls itself to calculate Fibonacci numbers.
  • The iterative approach is more efficient and has lower memory overhead.
  • Time complexity: O(n) for the iterative approach, exponential for the recursive approach.

Code Example:

function fibonacciIterative(n)
  if n <= 1 then
    return n
  end
  local fib = {0, 1}
  for i = 2, n do
    fib[i] = fib[i - 1] + fib[i - 2]
  end
  return fib[n]
end

The function 'fibonacciIterative' calculates the nth Fibonacci number using an iterative approach. It initializes a table 'fib' with the first two Fibonacci numbers. It then iterates from 2 to n, filling in the Fibonacci numbers by adding the last two elements of the table.

How would you implement a basic LRU cache in Lua?

Answer:

Implementing a Basic LRU Cache in Lua

  • Define a function 'LRUCache' that takes the maximum cache size as a parameter.
  • Initialize a table 'cache' to store key-value pairs.
  • Initialize a linked list 'recent' to maintain the order of recently accessed items.
  • Define a function 'get' to retrieve a value from the cache.
  • If the key is in the 'cache' table, move the corresponding node to the front of 'recent' and return the value.
  • Define a function 'set' to add a key-value pair to the cache.
  • If the cache is full, remove the least recently used item from 'cache' and 'recent'.
  • Add the new key-value pair to the 'cache' table and the front of the 'recent' list.
  • Time complexity: O(1) for 'get' and 'set'

Code Example:

function LRUCache(maxSize)
  local cache = {}
  local recent = {}
  return {
    get = function(key)
      local value = cache[key]
      if value then
        -- Move the accessed item to the front of 'recent'
        table.remove(recent, value)
        table.insert(recent, 1, value)
      end
      return value
    end,
    set = function(key, value)
      if #recent >= maxSize then
        local leastRecent = table.remove(recent)
        cache[leastRecent] = nil
      end
      cache[key] = value
      table.insert(recent, 1, key)
    end
  }
end

print(cache.get('a')) -- Output: 1
print(cache.get('c')) -- Output: 3
print(cache.get('d')) -- Output: nil (not in cache anymore)

The 'LRUCache' function returns an interface to interact with the cache. The 'get' function retrieves a value and moves the accessed item to the front of 'recent'. The 'set' function adds a key-value pair to the cache and removes the least recently used item if the cache is full. The example demonstrates cache operations and shows how least recently used items are evicted.
