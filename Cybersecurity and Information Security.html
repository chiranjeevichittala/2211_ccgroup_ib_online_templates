  • 3. Periodic Checks: Regularly re-calculate the hashes of the system files and compare them with the baseline hashes.
  • 4. Alert on Mismatch: Generate alerts if a file's hash differs from its baseline, indicating a change.
  • 5. Log the Changes: Keep a record of detected changes for further investigation.

Code Example:

import hashlib
import os

def file_hash(filename):
    """Calculate the SHA-256 hash of a file."""
    sha256_hash = hashlib.sha256()
    with open(filename, 'rb') as f:
        for byte_block in iter(lambda: f.read(4096),b""):
            sha256_hash.update(byte_block)
    return sha256_hash.hexdigest()

# Example usage

# Periodically check this hash against the current file hash

This script calculates the SHA-256 hash of a specified system file. By comparing this hash with a baseline, any changes to the file's content can be detected, signaling potential unauthorized modifications.

import ssl
import socket

def verify_ssl_certificate(domain):
    context = ssl.create_default_context()
    conn = context.wrap_socket(socket.socket(socket.AF_INET), server_hostname=domain)
    conn.settimeout(3.0)
    try:
        conn.connect((domain, 443))
        cert = conn.getpeercert()
        return cert
    except Exception as e:
        return str(e)
    finally:
        conn.close()

domains = ['example.com', 'anotherdomain.com']
for domain in domains:
    print(f'Verifying {domain}')
    cert_info = verify_ssl_certificate(domain)
    print(cert_info)

This script iterates over a list of domains, attempts to establish an SSL connection using the 'ssl' and 'socket' modules, and prints the certificate information for each domain. You can extend the script to check specific fields in the certificate, such as the expiry date.

This CSP directive allows framing of your content only by your own site ('self') and a specified trusted domain, adding an additional layer of protection against clickjacking.

  • 2. Track Requests: Use server-side logic to monitor and count requests from individual IP addresses.
  • 3. Enforce Limits: Temporarily block or slow down further requests once a threshold is exceeded.
  • 4. Notify Users: Optionally, inform users when they are nearing or have exceeded rate limits.

Code Example:

from flask import Flask, request, make_response
from functools import wraps
import time

app = Flask(__name__)

visitors = {}
def rate_limit(max_requests, time_window):
    def decorator(f):
        @wraps(f)
        def wrapped(*args, **kwargs):
            visitor = request.remote_addr
            if visitor not in visitors:
                visitors[visitor] = [time.time()]
            else:
                visitors[visitor].append(time.time())
            visitors[visitor] = [timestamp for timestamp in visitors[visitor] if time.time() - timestamp < time_window]
            if len(visitors[visitor]) > max_requests:
                return make_response('Rate limit exceeded.', 429)
            return f(*args, **kwargs)
        return wrapped
    return decorator
