Common Protocols in IoT Device Communication

Write a JavaScript function for a Node.js server to store IoT data into a MongoDB database.

Answer:

Storing IoT Data into MongoDB with Node.js

Code Example:

const MongoClient = require('mongodb').MongoClient;
const url = 'mongodb://localhost:27017';
const dbName = 'iot_data';
const client = new MongoClient(url, { useUnifiedTopology: true });

async function storeIoTData(data) {
  try {
    await client.connect();
    console.log('Connected correctly to server');
    const db = client.db(dbName);
    const collection = db.collection('data');
    const insertResult = await collection.insertOne(data);
    console.log('Inserted document:', insertResult);
  } catch (err) {
    console.error(err);
  } finally {
    await client.close();
  }
}

module.exports = { storeIoTData };

This JavaScript function initializes a MongoDB client connection, connects to a specified database, and inserts a data document into the 'data' collection. It uses async/await for handling asynchronous operations.

Question:

Write a function in C++ for Arduino to debounce a button used in an IoT device.

Answer:

Debouncing a Button in Arduino for IoT Devices

Code Example:

const int buttonPin = 2; // The pin the button is attached to
int buttonState = LOW; // Current state of the button
int lastButtonState = LOW; // Previous state of the button
unsigned long lastDebounceTime = 0; // the last time the output pin was toggled
unsigned long debounceDelay = 50; // the debounce time; increase if the output flickers

void setup() {
  pinMode(buttonPin, INPUT);
}

void loop() {
  int reading = digitalRead(buttonPin);
  if (reading != lastButtonState) {
    lastDebounceTime = millis();
  }
  if ((millis() - lastDebounceTime) > debounceDelay) {
    if (reading != buttonState) {
      buttonState = reading;
      if (buttonState == HIGH) {
        // put the button action code here
      }
    }
  }
  lastButtonState = reading;
}

This code snippet reads the state of a button and uses a debounce delay to ensure that the button's state is stable before triggering an action. This prevents false triggers due to the button's bounce.

Keywords: Arduino, C++, debounce, button, IoT Learn more

Code Example:

import pandas as pd
import numpy as np

# Example raw sensor data
data = pd.Series([20, 21, 19, 22, 30, 21, 20])

# Apply a moving average filter
window_size = 3
filtered_data = data.rolling(window=window_size).mean()
