Function to Check Binary Search Tree in Haskell

  • The function takes a binary tree as input, represented using a data type.
  • In the base case, when the tree is empty, it is considered a BST.
  • In the recursive case, the function checks whether the root value is greater than all elements in the left subtree and less than all elements in the right subtree.
  • It recursively applies the same property check to the left and right subtrees.
  • If all conditions are satisfied, the tree is a binary search tree; otherwise, it is not.

isBST :: Ord a => BinaryTree a -> Bool
isBST Empty = True
isBST (Node left value right) =
  all (< value) (elements left) && all (> value) (elements right) && isBST left && isBST right
  where
    elements Empty = []
    elements (Node l v r) = elements l ++ [v] ++ elements r

The function 'isBST' takes a binary tree 'tree' as input, where the tree is represented using a data type 'BinaryTree'. In the base case, when the tree is empty, it is considered a BST. In the recursive case, the function checks whether the root value is greater than all elements in the left subtree and less than all elements in the right subtree. It recursively applies the same property check to the left and right subtrees. The 'elements' function collects all elements of a tree in in-order traversal order. If all conditions are satisfied, the tree is a binary search tree; otherwise, it is not.

  • The function takes a comparison function that compares the lengths of two words.
  • The 'comparing' function from the 'Data.Ord' module can be used to generate the comparison function.
  • The 'sortBy' function sorts the words in ascending order of their length.
  • You can reverse the result to get the words sorted in descending order of length.

Code Example:

import Data.List
import Data.Ord

sortWordsByLength :: [String] -> [String]
sortWordsByLength words = sortBy (comparing length) words

The 'sortWordsByLength' function takes a list of words and sorts them by their length. It imports the required modules 'Data.List' and 'Data.Ord'. The 'sortBy' function is used with the 'comparing' function from 'Data.Ord' to compare the lengths of words. This function generates a comparison function that compares the lengths. The 'sortBy' function sorts the words in ascending order of length. To sort in descending order, you can reverse the result.
