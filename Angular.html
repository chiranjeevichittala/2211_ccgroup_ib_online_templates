Question:

Write a custom directive that changes the background color of an element when hovered.

Answer:

Writing a Custom Directive for Hover Background Color

  • Import the Directive decorator from '@angular/core'.
  • Use the @Directive decorator to define a custom directive.
  • Bind to the host element's mouseenter and mouseleave events using HostListener decorator.
  • Use the ElementRef to access the DOM element and Renderer2 to modify the styles.
  • Apply the desired background color when the mouseenter event is triggered.
  • Reset the background color when the mouseleave event is triggered.

Code Example:

import { Directive, ElementRef, Renderer2, HostListener } from '@angular/core';

@Directive({
  selector: '[appHoverHighlight]'
})
export class HoverHighlightDirective {

  constructor(private el: ElementRef, private renderer: Renderer2) {}

  @HostListener('mouseenter') onMouseEnter() {
    this.renderer.setStyle(this.el.nativeElement, 'background-color', 'lightblue');
  }

  @HostListener('mouseleave') onMouseLeave() {
    this.renderer.removeStyle(this.el.nativeElement, 'background-color');
  }
}

The example demonstrates creating a directive named 'appHoverHighlight' that changes the background color of an element when hovered. The HostListener decorators are used to listen for mouseenter and mouseleave events and apply or remove the background color using Renderer2.

  • Highlighting menu items when hovered for better user experience.
  • Adding visual feedback to buttons or clickable elements on hover.
  • Creating interactive elements that respond to user interactions.
  • Enhancing the look and feel of elements in a user interface.

Write a pipe to transform a string into title case in Angular.

Answer:

Creating a Title Case Pipe in Angular

  • Create a new file for the custom pipe (title-case.pipe.ts).
  • Import Pipe, PipeTransform from '@angular/core'.
  • Implement PipeTransform interface in the custom pipe class.
  • Define the 'transform' method to convert the string to title case.
  • Use the pipe in templates with the syntax: {{ value | titleCase }}.
  • Import and declare the custom pipe in the module.

Code Example:

import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'titleCase'
})
export class TitleCasePipe implements PipeTransform {

  transform(value: string): string {
    if (!value) return '';
    return value.split(' ').map(word => word.charAt(0).toUpperCase() + word.slice(1).toLowerCase()).join(' ');
  }
}

The example demonstrates creating a TitleCasePipe that implements PipeTransform. The 'transform' method takes a string as input and converts it to title case by splitting words, capitalizing the first letter of each word, and joining them back together.

  constructor(private userService: UserService) {}

  `

    return Math.ceil(this.items.length / this.itemsPerPage);

      this.currentPage--;
      this.updateDisplayedItems();
    }
  }

    if (this.currentPage < this.totalPages) {
      this.currentPage++;
      this.updateDisplayedItems();
    }
  }

  intercept(req: HttpRequest<any>, next: HttpHandler) {
    if (!req.headers.has('cache-control')) {
      const cachedResponse = this.cache.get(req.url);
      if (cachedResponse) {
        return of(cachedResponse.clone());
      }

      return next.handle(req).pipe(
        tap(event => {
          if (event instanceof HttpResponse) {
            this.cache.set(req.url, event.clone());
          }
        })
      );
    }

    return next.handle(req);
  }
}

This interceptor caches HTTP responses in a map and serves cached responses for subsequent requests. Fresh responses are stored in the cache when fetched from the server.

Sharing Data Between Unrelated Components using BehaviorSubject

import { Injectable } from '@angular/core';
import { BehaviorSubject, Observable } from 'rxjs';

@Injectable({ providedIn: 'root' })
export class DataSharingService {
  private dataSubject = new BehaviorSubject<any>(null);

  setData(data: any): void {
    this.dataSubject.next(data);
  }

  getData(): Observable<any> {
    return this.dataSubject.asObservable();
  }
}

This service demonstrates data sharing using a BehaviorSubject. It provides methods to set and get data. Components can subscribe to the Observable returned by getData() to receive updates.
