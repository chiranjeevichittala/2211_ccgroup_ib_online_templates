Write a program to find the product of two matrices in JavaScript.

Answer:

Product of Two Matrices in JavaScript

  • 1. Check if the number of columns in the first matrix is equal to the number of rows in the second matrix. If not, matrix multiplication is not possible.
  • 2. Create a new matrix to store the product with the appropriate dimensions (rows x columns).
  • 3. Use nested loops to perform the multiplication operation.
  • 4. Multiply each element of the first matrix with the corresponding element of the second matrix and accumulate the result in the new matrix.

Code Example:

function matrixProduct(matrix1, matrix2) {
    const rows1 = matrix1.length;
    const columns1 = matrix1[0].length;
    const rows2 = matrix2.length;
    const columns2 = matrix2[0].length;

    if (columns1 !== rows2) {
        return "Matrix multiplication is not possible.";
    }

    const resultMatrix = new Array(rows1);
    for (let i = 0; i < rows1; i++) {
        resultMatrix[i] = new Array(columns2).fill(0);
    }

    for (let i = 0; i < rows1; i++) {
        for (let j = 0; j < columns2; j++) {
            for (let k = 0; k < columns1; k++) {
                resultMatrix[i][j] += matrix1[i][k] * matrix2[k][j];
            }
        }
    }

    return resultMatrix;
}

const matrix1 = [
    [1, 2],
    [3, 4],
];

const matrix2 = [
    [5, 6],
    [7, 8],
];

console.log("Product of Matrices:", matrixProduct(matrix1, matrix2));

In this example, the "matrixProduct" function takes two matrices as input and performs matrix multiplication using nested loops. The function first checks if the matrices can be multiplied by comparing their dimensions. It creates a new matrix "resultMatrix" with the appropriate dimensions to store the product. The nested loops perform the multiplication operation, where each element of the first matrix is multiplied with the corresponding element of the second matrix and accumulated in the "resultMatrix." The function is called with sample matrices, and the product is displayed using console.log.

Question:

Write a program to find the height of a binary tree in JavaScript.

Answer:

Find the Height of a Binary Tree in JavaScript

  • 1. To find the height of a binary tree, you can use a recursive approach.
  • 2. The height of an empty tree is 0.
  • 3. The height of a non-empty tree is the maximum of the heights of its left and right subtrees, plus 1.
  • 4. Recursively calculate the height of the left and right subtrees and return the maximum height plus 1.

Code Example:

// Node class representing a binary tree node
class Node {
    constructor(data) {
        this.data = data;
        this.left = null;
        this.right = null;
    }
}

// Function to find the height of a binary tree
function findBinaryTreeHeight(root) {
    if (!root) {
        return 0;
    }
    const leftHeight = findBinaryTreeHeight(root.left);
    const rightHeight = findBinaryTreeHeight(root.right);
    return Math.max(leftHeight, rightHeight) + 1;
}

// Create a sample binary tree
const root = new Node(10);
root.left = new Node(5);
root.right = new Node(15);
root.left.left = new Node(3);
root.left.right = new Node(7);
root.right.right = new Node(20);

const height = findBinaryTreeHeight(root);
console.log("Height of the binary tree:", height);

In this example, a class "Node" is defined to represent a binary tree node. The class has properties for data, left child, and right child. A function "findBinaryTreeHeight" is defined to find the height of a binary tree. The function takes the root of the tree as input. It first checks if the root is null, indicating an empty tree, and returns 0. Otherwise, it recursively calculates the height of the left and right subtrees using the "findBinaryTreeHeight" function. It returns the maximum height of the two subtrees plus 1, representing the height of the current node. The sample binary tree is created, and its height is calculated using the function. The result is displayed using console.log.

What is the prototype in JavaScript?

Answer:

Prototype in JavaScript

  • 1. Prototype Chain: Every object in JavaScript has a prototype, and objects inherit properties and methods from their prototype's prototype.
  • 2. Object Creation: When a new object is created using a constructor function or object literal, it inherits properties and methods from its constructor's prototype.

function Person(name, age) {

In this example, we define a constructor function 'Person' that takes 'name' and 'age' as parameters and assigns them to the instance. We then add a 'greet' method to the 'Person.prototype', which is accessible by all instances of 'Person' through the prototype chain.

Real-time Usage:

  • Prototypes are used in object-oriented programming to create classes and define shared behavior for objects.
  • Inheritance is achieved in JavaScript through the prototype chain, where objects inherit from their prototypes.
  • Modifying a prototype is a common way to extend the functionality of built-in JavaScript objects, such as adding custom methods to arrays.

Question:

Write a program to merge two sorted linked lists in JavaScript.

Answer:

Merge Two Sorted Linked Lists in JavaScript

  • 1. Initialize a dummy node to serve as the new head of the merged list.
  • 2. Initialize two pointers to traverse the input lists, one for each list.
  • 3. Compare the values of the nodes pointed by the two pointers.
  • 4. Attach the node with the smaller value to the merged list and move the corresponding pointer to the next node.
  • 5. Repeat the process until you reach the end of either of the input lists.
  • 6. Attach the remaining nodes of the non-empty list to the merged list.
  • 7. Return the head of the merged list.

Code Example:

// Definition of a linked list node
class Node {
    constructor(data) {
        this.data = data;
        this.next = null;
    }
}

// Function to merge two sorted linked lists
function mergeSortedLists(l1, l2) {
    let dummy = new Node(-1);
    let current = dummy;
    while (l1 && l2) {
        if (l1.data < l2.data) {
            current.next = l1;
            l1 = l1.next;
        } else {
            current.next = l2;
            l2 = l2.next;
        }
        current = current.next;
    }
    current.next = l1 || l2;
    return dummy.next;
}

// Create two sorted linked lists
const list1Node1 = new Node(1);
const list1Node2 = new Node(3);
const list1Node3 = new Node(5);

list1Node1.next = list1Node2;
list1Node2.next = list1Node3;

const list2Node1 = new Node(2);
const list2Node2 = new Node(4);
const list2Node3 = new Node(6);

list2Node1.next = list2Node2;
list2Node2.next = list2Node3;

const mergedList = mergeSortedLists(list1Node1, list2Node1);

// Print the merged linked list
let result = "";
let current = mergedList;
while (current) {
    result += current.data + " -> ";
    current = current.next;
}
console.log("Merged Linked List:", result.slice(0, -4));

In this example, a class "Node" is defined to represent a node in a linked list. The "mergeSortedLists" function takes two head nodes of sorted linked lists as input and returns the head node of the merged list. It uses a dummy node to facilitate the merging process. The "current" pointer is used to keep track of the merged list. The function iterates through both lists, comparing the data of nodes pointed by the two pointers. The node with the smaller value is attached to the merged list, and the corresponding pointer is moved to the next node. When either of the input lists reaches its end, the remaining nodes of the other list are attached to the merged list. The function is called with two sample sorted linked lists, and the merged linked list is displayed using console.log.

    let currentNode = head;
    let prevNode = null;

    // If the key is found in the head node, update the head
    if (currentNode !== null && currentNode.data === key) {
        head = currentNode.next;
        return head;

    }

}

What is the 'some' method in JavaScript?

Answer:

Some Method in JavaScript

  • 1. Syntax: The syntax of the some method is as follows: array.some(callbackFn(element, index, array))
  • 2. Callback Function: The callback function is called for each element of the array and can take up to three arguments: element (current value), index (current index), and array (the original array being processed).
  • 3. Return Value: The some method returns a Boolean value (true or false). It returns true if at least one element satisfies the condition specified in the callback function. Otherwise, it returns false.

Code Example:

const numbers = [5, 8, 12, 4, 7];
const result = numbers.some((num) => num > 10);
console.log(result); // Output: true

In this example, the some method is used to check if any number in the "numbers" array is greater than 10. The callback function checks whether each element (num) is greater than 10, and the some method returns true as soon as it finds the first element that satisfies the condition (num > 10).

const fruits = ["apple", "banana", "orange"];
const searchSubstring = "an";
const result = fruits.some((fruit) => fruit.includes(searchSubstring));
console.log(result); // Output: true

In this example, the some method is used to check if any string in the "fruits" array contains the substring "an". The callback function uses the includes method to check each string (fruit) for the presence of the substring, and the some method returns true as soon as it finds the first element that contains the substring.
